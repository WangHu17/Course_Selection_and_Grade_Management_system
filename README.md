# 开发环境
- 数据库：MySql 8.0.22
- 服务器：apache-tomcat-9.0.41
- 编译器：IntelliJ IDEA 2021.1.1
- 浏览器：Chrome
# 技术栈
- 后端技术：java、sql、jsp
- 后端框架：SSM（spring+springmvc+mybatis）
- 前端技术：HTML、CSS、JavaScript、Ajax
- 前端框架：bootstrap-3.3.7-dist、jquery
- 项目管理工具：maven
